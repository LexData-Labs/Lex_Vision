# Contributing to Lex Vision

Thank you for your interest in contributing to Lex Vision!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone <your-fork-url>`
3. Create a virtual environment: `python -m venv venv`
4. Install dependencies: `pip install -r requirements.txt`
5. Install frontend dependencies: `cd frontend && npm install`
6. Create a new branch: `git checkout -b feature/your-feature`

## Code Structure

- **Backend**: All code in `backend/backend.py` (single merged file)
- **Frontend**: React + TypeScript in `frontend/` directory
- **Data**: Employee faces in `data/employee_faces/`

## Submitting Changes

1. Make your changes
2. Test thoroughly
3. Commit with clear messages
4. Push to your fork
5. Create a Pull Request

## Code Style

- Python: Follow PEP 8
- TypeScript: Follow ESLint rules
- Use meaningful variable names
- Add comments for complex logic
